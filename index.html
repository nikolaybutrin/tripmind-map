Регистрация и авторизация
Entry
Быстрый вход через Google, Apple, VK ID, email. OAuth 2.0 + JWT. 152-ФЗ: ПД в Yandex Cloud (ФСТЭК УЗ-1). Не более 3 экранов до первого маршрута.
OAuth 2.0: Google / Apple / VK ID / email
Восстановление пароля и 2FA (масштабирование)
Solo Auto Group — Нет AI
← Лендинг → Онбординг

Wizard: стиль, интересы, бюджет, роль

Зона 2
AI-Планирование маршрута
Ключевая ценность продукта — от AI-генерации до финального маршрута на карте
MVP
🤖
AI-Маршрутизатор (Core)
MVP Core
Главный модуль. Генерирует маршрут по дням с POI. ML-модуль ранжирования POI + rule-based оптимизация порядка точек (TSP-эвристика).
ML: ранжирование POI по профилю (acceptance rate ≥35%)
Rules: оптимизация маршрута (расстояния, часы работы)
Стек: LangGraph + OR-Tools (TSP/VRP) + Mistral/Llama 3 self-hosted
Вход: профиль → Выход: маршрут → чек-лист, хаб
Solo Auto Group ⚡ML + Rules
← Онбординг → Редактор, Чек-лист, Персонализация, Разведка
MVP
🧠
Модуль AI-персонализации
ML Core
Основная ML-точка проекта. Гибридный CF+CBF (Precision@10=0,312 на TripAdvisor). Self-hosted на Yandex Cloud. Ранжированные рекомендации с объяснениями и feedback-циклом.
ML: collaborative filtering по профилю и истории
Feedback-цикл: «подходит» / «не подходит»
Rules: шаблонные объяснения решений AI
Метрика: CTR ≥35% к 6-й поездке
Solo Auto Group ⚡ML
← Профиль, История → Маршрутизатор, Разведка, Чек-лист

Модуль разведки и открытия мест
Discovery

ML: рекомендации POI по профилю (масштабирование)
Rules MVP: фильтрация по рейтингу, расстоянию, часам

Режим «Авто-маршрут»
Auto-Mode
Расчёт плеч, ночёвок, бензина. Rule-based алгоритм + API карт и АЗС. ML не требуется.
